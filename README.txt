1. Ceci est un application console de restauration qui permet au utilisateur de passer des commandes, 
   faire des reservation. cette application peut aussi montrer des differents restaurants et leur menu.

2. Il est à noté que les liste des restaurant et les differents menu sont contenu dans des fichiers txt pour une plus simple manipulation 
   tous present dans le dossier du projet et donc il faudrait modifier le chemin d'accèss vers ces fichiers pour pouvoir lancer le program SVP.

3. Pour les differentes fonctionnalités de cette application, tous sont fonctionnels et les resultats se font dans des fichiers. 
   example:pour un utilisateur qui fait un reservation, ses données sont pris et enregistrer dans le fichier "fich resev" pour l'ecture par l'admin.
